Instructions

  1. Navigate to one of the exercises (listed below)
  2. clone and cd to the exercise repo, if you haven't done so already
  3. Run the contained go script by executing ./go.
    This sets up the required Git hooks, resets the repo, and starts the exercise.
  4. Follow the instructions that appear in your browser and terminal
If you ever get yourself into too much trouble or want to restart the exercise, just run the go script and everything will be reset, fresh and ready to go!

Exercises

  1. First Steps: Adding a new file to remote version control
  2. Editing a version-controlled file
  3. Merge conflict!
